Usage: coss4 [option]
Options:
    -b DIRECTORY, --build DIRECTORY
                        Build the coss4 project and package it, then move the
                        generated jar package or war package to the specified
                        location. If DIRECTORY is not specified, only the
                        build is executed and the generated jar package or
                        war package will not be moved.
    -c, --clean         Clear the target files generated by jar or war
                        package, etc. If directory is specified when the
                        build command is executed, the execution of the clean
                        command will not affect the files in that directory.
    -h, --help          Print this message and exit.
    --init DATABASE_NAME
                        Initialize the specified database.
    -i, --install       Run maven install for coss4 server, package and
                        install the project into local Maven repository.
    -t, --test          Run unit tests for coss4 with JUnit.
    -v, --version       Print the version number of coss4 and exit.

This program built for Java jdk-8u241
Report issues to <https://github.com/cambridgejames/coss4-server/issues>
